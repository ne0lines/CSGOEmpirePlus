# CSGOEmpire Plus Goal

CSGOEmpire Plus enriches CSGOEmpire item pages with deeper skin market analysis than CSGOEmpire shows by default.

The extension should let a user add their own Pricempire API key, then show a fixed analysis panel on each CSGOEmpire item page with:

- Current CSGOEmpire item data.
- Pricempire trader insights.
- Pricempire historical chart data.
- Historical CSGOEmpire sales as events on the chart.
- CSGOEmpire exchange-rates for currency conversion.

API keys stay local in Chrome storage and are only used by the extension background service worker for Pricempire requests.
